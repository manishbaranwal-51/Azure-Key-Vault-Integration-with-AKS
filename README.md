# Azure-Key-Vault-Integration-with-AKS

📘 Overview
This project automates the creation and deployment of a Python-based Azure Function App using Azure CLI commands.
It demonstrates how to provision all the necessary resources — Resource Group, Storage Account, and Function App — in one go.

This is part of my #100DaysOfCloud / Azure Practice Series, focusing on serverless compute automation.

⚙️ Tech Stack
Platform: Microsoft Azure
Language: Bash (for CLI automation)
Runtime: Python 3.9
Service: Azure Function App (Functions v4)
OS Type: Linux
Tools: Azure CLI, VS Code, GitHub

🧩 Project Structure
│
├── create-function.sh           # Azure CLI script to create all resources
├── README.md                    # Documentation (this file)

💡 Purpose
To understand and automate the deployment of a serverless Python Function App on Azure using CLI commands —
no manual portal configuration needed.
